Motion.JS, For mobile web applications.

Motion.JS is a swift javascript framework,which includes a bundle of useful components leveraging the cool new features from Html5. Like the name 'Motion', most Motion.JS components are behavior focused, it's a piece of cake for you to custom their UI styles. In addition, Motion.JS is zepto based, so you don't need to warry about manipulating dom elements at any time when using it.
